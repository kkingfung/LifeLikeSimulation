# 第2夜: 炎の証言

## 概要

**シナリオID:** `night_02`
**タイトル:** 炎の証言
**時間:** 23:00 - 01:00（ゲーム内時刻）
**難易度:** 4/5
**ステータス:** ✅ 設計完了・データ作成完了

---

## 真実

Night01の翌日（24時間後）。組織のフロント企業「アラートセキュリティ」の倉庫で火災が発生。**原因は100%事故（危険物の不適切な保管による自然発火）。証拠隠滅ではない。**

**重要:** プレイヤーは「組織が証拠を隠滅したのでは？」と疑う可能性があるが、実際には偶然の事故。ただし、組織のフロント企業であることは事実であり、Night04で目撃される社用車と同じ車が火災現場でも目撃されている。

高齢者の吉田和子は林遥の隣人。遥と美咲を「近所の若いお母さんと子供」として知っているが、遥の死はまだ知らない。最近見かけないことを心配している。

公園で通報された「不審者」の正体は、遥の友人・渡辺真理。遥の死を知り、行方不明の美咲を探している。Night02では直接登場せず、Night03への伏線。

工藤主任はNight01の報告書から「不要な詳細」を削除するよう圧力をかけてくる。

---

## 設計原則

Night02は「別の角度から見る」フェーズ:
- Night01とは**直接関係ないように見える**火災事件
- 複数の目撃者が**同じ断片を見ている**
- プレイヤーが**接続に気づけるか**がテーマ
- Night03への伏線（渡辺真理、アラートセキュリティ）

### Night01との差別化

| Night01 | Night02 |
|---------|---------|
| 被害者を救えるかどうか | 情報を集められるかどうか |
| 即座の決断が求められる | 断片を繋げる観察が求められる |
| 感情的な通話が多い | 事実報告型の通話が多い |
| 組織が直接介入 | 組織の影が間接的に見える |

## 発信者一覧

### 1. 吉田和子 - 「高齢者」【新規・重要】
- **通話ID:** `call_fire_elderly` / `call_elderly_callback`
- **タイトル:** 「火事です！」「あの...思い出したことがあって」
- **時刻:** 23:05 / 23:55（フラグ依存）
- **役割:** 火災第一報 / Night01との接点

**プロフィール:**
- 72歳、一人暮らし
- 夫を5年前に亡くし、子供は遠方
- 林遥の隣人として、遥と美咲を気にかけていた
- 夜間の病院帰りに火災を目撃

**組織との関係:** なし

**【重要な隠し設定】美咲を自宅に招いている:**
- 遥が最近見かけないことを心配し、美咲を自宅に招いた
- 美咲は吉田和子の家で安全に過ごしている
- 真理も組織もこの事実を知らない
- Night05以降で重要な伏線になる

**彼女が知っていること:**
- 爆発音の後に火災が発生した
- 火災前に白いバンが倉庫に入っていくのを見た
- 車には「アラート」と書いてあった気がする（後で思い出す）
- 近所の「若いお母さん」（林遥）が最近見かけない
- **美咲が自宅にいること（プレイヤーには伝えない）**

**彼女が知らないこと:**
- 林遥が死んだこと
- 組織の存在
- 倉庫と組織の関係
- 美咲が「探されている」こと

**Night02での役割:**
- 火災の第一報（パニック状態）
- 車について質問されると、後で折り返して詳細を教える
- 「近所の若いお母さん」として林遥に言及（プレイヤーがNight01と接続できるかの伏線）

## 重要な設計ノート

### 火災について

- 火災そのものは解決される（消防が対応）
- プレイヤーの選択は「情報をどれだけ集めたか」に影響
- 爆発音と車両情報が重要な証拠

### 吉田和子について

- Night01の林遥との唯一の接点
- プレイヤーが「若いお母さん」への言及に気づけるかがポイント
- 車について質問すれば、折り返しでアラートセキュリティの名前を得られる

### 渡辺真理について

- Night02では「不審者」としてのみ登場
- 正体はプレイヤーには分からない
- Night03で本人として電話してくる予定
