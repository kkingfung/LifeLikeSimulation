## あらすじ

真理が連れ去られた夜。警察は行方不明届の確認に動き、吉田は真理が話していたことを思い出す。
工藤からは不穏な噂が届く。そして、深入りしたオペレーターには組織からの警告が届く。

---

## 真実

真理は組織に連れ去られた。警察は行方不明届を受理したが、誠和製薬との繋がりには気づいていない。

しかし、林健二と中村浩二の事件に同じ人間が関わっているという噂が警察内部で流れている。

組織はオペレーターの動きを監視しており、深入りした者には警告を送る。

美咲は吉田の家で安全に守られている（yoshida_will_warn_mariルートの場合）。

## エンディング

### 真実を追う者 (TruthSeeker)
> 警察に全てを話した。組織の警告を無視した。
>
> 林健二、中村浩二、渡辺真理——すべてが繋がっている。誠和製薬という巨大な影。
>
> 組織は私を監視している。でも、もう止まれない。
>
> 美咲ちゃんを守らなければ。真実を明らかにしなければ。

### 慎重な知識 (InformedCaution)
> 多くのことを知った。遥と真理が姉妹だったこと。健二が何かを知っていたこと。
>
> でも、組織の警告を受け入れた。「賢い選択」だと言われた。
>
> 知っている。でも、動けない。それが今の私にできる精一杯。

### 沈黙の証人 (SilentWitness)
> いくつかの断片が繋がり始めた。でも、全容はまだ見えない。
>
> 警察も慎重に動いている。工藤も何かを知っている。
>
> 私は何をすべきなのか。まだ、答えが見つからない。

### 無知な生存者 (UnawareSurvivor)
> 夜が明けた。渡辺真理は行方不明のまま。
>
> 何が起きているのか、よく分からない。でも、それでいいのかもしれない。
>
> 知らない方が、安全だから。

## 分岐ポイントまとめ

### 組織の警告への対応（重要な分岐）

```
組織からの警告電話（dispatched_to_mari_houseフラグがある場合）
    │
    ├─ 「分かりました」と従う
    │   └─ submitted_to_warning = true
    │       └─ Night09: 消極的な立場
    │           └─ Night10: 傍観者エンディングへ
    │
    └─ 「脅しには屈しない」と拒否
        └─ defied_organization = true
            └─ オペレーターも危険に
                └─ Night09: 積極的な立場
                    └─ Night10: 真実解明ルートへ
```

### 警察への協力度

1. **全てを話す**
   - 真理の電話、聞こえた声、ナンバープレート、林健二との繋がり
   - → `police_knows_connection`（Night09で有利）

2. **部分的に話す**
   - 一部の情報のみ
   - → 中程度の協力

3. **最低限の対応**
   - 証拠フラグが少ない
   - → Night09で不利

### 吉田の電話での情報収集

- 姉妹の話を聞く → `haruka_mari_sisters_confirmed`
- 健二の話を聞く → `kenji_knew_something`
- 誠和製薬の話を聞く → `seiwa_mentioned_by_mari`
- 怖がっていた話を聞く → `mari_was_afraid`
