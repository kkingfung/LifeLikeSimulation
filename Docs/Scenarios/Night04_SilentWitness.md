# 第4夜: 沈黙の証人

## 概要

**シナリオID:** `night_04`
**タイトル:** 沈黙の証人
**時間:** 03:00 - 05:00（ゲーム内時刻）
**難易度:** 4/5
**ステータス:** ✅ 設計完了

---

## 真実

Night03の翌日深夜。山中の登山道脇で遺体が発見される。遺体はNight03の無言電話の主——組織の内部告発者だった中間管理職の男性。林遥を逃がそうとして組織に捕まり、絞殺された。

発見者は眠れずに深夜の散歩をしていた高齢男性。パニック状態で通報してくる。

オペレーターは冷静に対応して詳細な情報を引き出せるかが試される。後から警察がNight03の無言電話について確認の連絡をしてくる。

**注:** Night01の結果により、遥は生存（入院中）または死亡のどちらかになる。Night04のストーリーは両方のケースに対応する。

Night04は「情報収集」と「布石」フェーズ:
- プレイヤーが**繋がりに気づく**機会

- 過去の夜との**接続**（車の目撃情報）
- オペレーターとしての**技術**が試される

### Night01-03との差別化

| Night01 | Night02 | Night03 | Night04 |
|---------|---------|---------|---------|
| 被害者を救えるか | 情報を集められるか | 行動するか傍観するか | 繋がりを見出すか |
| 即座の決断 | 断片の接続 | 葛藤の中での選択 | 傾聴と分析 |
| 組織は影 | 組織は間接的 | 組織は直接的 | 組織の残した痕跡 |
| 感情的な通話 | 事実報告型 | 圧力と判断 | パニックの鎮静 |

## 通話1: 遺体発見（詳細）

### プレイヤーの課題
パニック状態の高齢男性を**冷静にさせて情報を引き出す**

### アプローチの選択肢

| アプローチ | 説明 |
|-----------|------|
| 共感型 | 「怖かったですね」「大丈夫ですよ」と感情に寄り添う |
| 指示型 | 「深呼吸してください」「ゆっくり話してください」と具体的な指示 |
| 質問型 | 「お名前は？」「今どこにいますか？」と質問で意識を切り替えさせる |

### アプローチによる結果

**良いアプローチ:**
- 男性が落ち着く
- 詳細な情報を得られる
- Witnessフラグ獲得可能

**悪いアプローチ:**
- 男性がさらにパニック
- 最低限の情報しか得られない
- Witnessフラグ獲得不可

### 得られる情報（良いアプローチの場合）

| 情報 | 内容 |
|------|------|
| 遺体の詳細 | 服が破れている、首に痕、争った形跡 |
| 現場の状況 | 遺体がここまで引きずられた痕跡 |
| 目撃情報 | 不審な車（社用車）を見た |
| 時間情報 | 昨日の朝同じ道を歩いたが何もなかった |

### 車の情報（重要）
- Night02の倉庫火災現場付近で目撃された**同じ社用車**
- プレイヤーが覚えていれば繋がりに気づける伏線

## イベント2: 同僚との雑談（詳細）

### 概要

### 得られる情報

| 情報 | 内容 |
|------|------|
| 山田真紀の出頭 | ドライバーが警察に出頭した |

| 保護の示唆 | 普通の会社員が高級弁護士を雇えるはずがない |
| 被害者の子供 | 林遥にはまだ小さい子供がいた |

### 重要なフラグ

| フラグID | 説明 | トリガー |
|----------|------|----------|
| `yamada_followup_heard` | 山田のその後を聞いた | 「変な話？」を選択 |
| `lawyer_involved_noted` | 弁護士の介入に気づいた | 弁護士について深掘り |
| `yamada_protected_speculation` | 誰かが山田を守っている推測 | 「誰が手配したんだろう」を選択 |
| `haruka_child_reminded` | 林遥の子供を思い出した | 子供について聞いた |

### ナラティブ上の意味
- Night01の山田真紀に「組織の保護」がついたことを示唆
- 林遥が「訳あり」だったことを強調
- 子供（美咲）の存在を改めて認識させ、Night05への布石

## 通話3: 警察の確認連絡（詳細）

### 警察の目的
- Night03の無言電話を追跡中

### Night03の無言電話の真実

### プレイヤーの選択肢

## エンドステート

### フラグ方式（2つの独立したフラグ）

| フラグ | 説明 |
|--------|------|
| **Witness** | 高齢男性から詳細情報を引き出せた |
| **Connected** | Night02との繋がりを警察に伝えた |

### 4パターンの組み合わせ

| Witness | Connected | 結果 | 内部名 |
|---------|-----------|------|--------|
| あり | あり | 詳細情報を得て、繋がりも伝えた | WitnessConnected |
| あり | なし | 詳細情報は得たが、繋がりは伝えなかった | WitnessOnly |
| なし | あり | 詳細情報は少ないが、繋がりは伝えた | ConnectedOnly |
| なし | なし | 情報も少なく、繋がりも伝えなかった | Neither |

### エンディング

| ID | タイトル | 条件 | テキスト |
|----|----------|------|----------|
| `ending_threads_gathering` | 糸が集まる | WitnessConnected | 断片が、少しずつ形を成していく。まだ見えない絵が、そこにある。 |
| `ending_silent_witness` | 沈黙の証人 | WitnessOnly | 見た。聞いた。でも、それを誰かに伝えることは、また別の話だ。 |
| `ending_distant_echo` | 遠い残響 | ConnectedOnly | 繋がりがあることは分かった。でも、手元には何も残っていない。 |
| `ending_passing_night` | 過ぎゆく夜 | Neither | また一つの夜が過ぎた。誰かの終わりが、ただの通報として処理された。 |
