## あらすじ

中村浩二が自宅で遺体で発見される。自殺に見せかけられているが、遺書はない。
同時に、組織が真理に迫る。美咲は吉田の家に預けられている。

---

## 真実

中村浩二は誠和製薬について騒ぎすぎたため、組織に口封じされた。首吊りに見せかけた殺害。遺書がないのは不自然。

真理は危険を感じて美咲を吉田に預けていた（`yoshida_will_warn_mari`フラグがある場合）。

組織は真理を連れ去るが、美咲は吉田の家で無事。吉田は美咲を守るために美咲のことを誰にも言わない。

**重要**: 真理は必ず連れ去られる。美咲の運命のみがNight06の選択で決まる。

## 分岐ポイントまとめ

### 美咲の運命を決める分岐（Night06で決定）

```
Night06: 吉田の通報への対応
    │
    ├─ パトロール派遣 + 真理に警告を勧める
    │   └─ yoshida_will_warn_mari = true
    │       └─ Night07: 真理が美咲を吉田に預ける
    │           └─ misaki_safe_at_yoshida = true
    │               └─ 美咲は安全
    │
    └─ 放置 / 警告なし
        └─ yoshida_will_warn_mari = false
            └─ Night07: 美咲は真理と一緒にいる
                └─ 組織が来たとき...
                    ├─ 真理が電話中に美咲を隠す
                    │   └─ misaki_fate_unknown = true
                    │
                    └─ 真理と美咲が一緒に連れ去られる
                        └─ misaki_taken = true
                            └─ 最悪の結末へ
```

### Night07での分岐

1. **中村の遺体発見への対応**
   - 詳細を聞く → 証拠フラグ獲得
   - 最低限の対応 → 証拠が少ない

2. **真理の緊急電話への対応**
   - 即座に警察派遣 → `dispatched_to_mari_house`（Night08で組織に目をつけられる）
   - 躊躇 → 派遣が遅れる

3. **吉田の報告への対応**
   - 詳細を聞く → 車のナンバー、方向などの証拠
   - 美咲について聞く → 吉田は答えない（美咲を守るため）
