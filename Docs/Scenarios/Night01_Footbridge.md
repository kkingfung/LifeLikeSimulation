# 第1夜: 午前2時の歩道橋

## 概要

**シナリオID:** `night_01`
**タイトル:** 午前2時の歩道橋
**時間:** 02:17 - 03:25（ゲーム内時刻）
**難易度:** 3/5
**ステータス:** ✅ 設計完了・データ作成完了

---

## 真実

林遥（はやし はるか）、27歳。シングルマザー。
深夜02:17頃、歩道橋の近くで車に轢かれた。

ドライバーの山田真紀（やまだ まき）は現場から逃走したが、罪悪感から緊急通報センターに電話をかけてくる。

林遥には5歳の娘・美咲がいる。美咲の行方は第1夜では不明のまま。

「家族」として電話してくる男は林遥の兄ではない。組織の人間であり、林遥の状況を探っている。

---

## 設計原則

Night01は「基本を教える」フェーズ:
- プレイヤーに**断片的な情報が集まる**ことを体験させる
- **言葉の選択が結果を変える**ことを示す
- **後から振り返ると意味が変わる**伏線を植える
- 組織の存在を**微かに示唆**（明確には見せない）

### コアメカニクス

| メカニクス | Night01での役割 |
|-----------|-----------------|
| 情報の非対称性 | プレイヤーはどの発信者よりも多くを知る |
| 道徳的圧縮 | すべての合理的な選択には結果がある |
| 遅延した啓示 | 真実は説明ではなく蓄積で浮かび上がる |
| 言葉の力 | 言葉は残り、トーンは積み重なる |

## エンドステート

### 被害者生存計算

| 条件 | 結果 |
|------|------|
| `emergency_dispatched` かつ `dispatch_time <= 02:49` | 生存 |
| `emergency_dispatched` かつ `dispatch_time > 02:49` | 死亡（間に合わず） |
| `!emergency_dispatched` | 死亡（未発見） |

### 5つのエンドステート

| 状態 | 条件 | 説明 |
|------|------|------|
| **Exposed（露出）** | disclosure_score >= 4, escalation_score >= 3 | 真相を明らかにした |
| **Contained（封じ込め）** | escalation_score >= 0, disclosure_score <= 2 | 対応したが抑制的 |
| **Complicit（共犯）** | reassurance_score >= 5, yamada_protected | ドライバーを守った |
| **Absorbed（吸収）** | escalation_score == 0, disclosure_score <= 1 | 何もしなかった |
| **Flagged（要注意）** | system_trust <= -3 OR supervisor_flagged | システムに目をつけられた |

### エンディング

| ID | タイトル | エンドステート + 生存 |
|----|----------|----------------------|
| `ending_dawn_light` | 夜明けの最初の光 | Exposed + 生存 |
| `ending_long_road` | 長い帰り道 | Contained + 生存 |
| `ending_silent_partner` | 沈黙の共犯者 | Complicit |
| `ending_weight_of_silence` | 沈黙には重さがある | Absorbed |
| `ending_weight_of_knowing` | 知ることの重さ | Exposed + 死亡 |
| `ending_just_another_night` | ただの一夜 | Contained/Flagged + 死亡 |
