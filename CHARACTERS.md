# Operator: Night Signal - キャラクターガイド

## 概要

本作に登場する全キャラクターの詳細情報。
キャラクターは大きく分けて「被害者側」「組織側」「一般市民」「オペレーター関係者」に分類される。

## 被害者・関係者

### 林 遥（はやし はるか）

| 項目 | 内容 |
|------|------|
| 年齢 | 27歳 |
| 職業 | 誠和製薬 研究員（逃走中） |
| 登場 | Night01（事故被害者として言及）|
| 状態 | Night01の選択により生存/死亡 |

#### 背景
誠和製薬の研究員として働いていた際、会社の違法な薬品試験と副作用データの隠蔽を発見。
証拠をUSBに保存し、娘の美咲に「キーホルダー」として持たせた。
組織から逃走中、疲労困憊のドライバー（山田美咲）に轢かれる。

#### 人物像
- 5歳の娘・美咲の母親
- 高校時代からの親友・渡辺真理に美咲を託していた
- 誠和製薬の研究主任・林健二（本物）の直属の部下だった
- 組織の末端工作員として使われていたが、真実を知り逃走を決意

|------|------|

| 職業 | - |
| 登場 | Night05（電話）、以降は吉田の家 |
| 状態 | 物語のキーパーソン |

#### 背景

## オペレーター関係者

### 工藤 真司（くどう しんじ）

| 項目 | 内容 |
|------|------|
| 年齢 | 45歳 |
| 職業 | 緊急通報センター 当直主任 |
| 登場 | 全夜（直接/チャット） |
| 組織関係 | system_aligned |

#### 性格パラメータ
- 正直さ: 50-60
- 安定性: 85-90
- 協力性: 40-50
- 攻撃性: 20-30

#### 背景
10年のベテラン当直主任。システムの代弁者。
組織との直接の関係はないが、「上からの圧力」を感じている。
情報の封じ込めを重視し、自分の立場を守りつつ部下も守りたいと思っている。
Night02で報告書から「不要な詳細」を削除するよう婉曲に圧力をかける。

#### 声の特徴
落ち着いた低い声。権威的だが穏やか。

#### 役割の変化
- Night01-02: 報告書の管理、圧力の伝達

- Night06以降: チャットで情報を伝える（遺体の身元判明など）

## 組織関係レベル

キャラクターと組織の関係を示すパラメータ:

| レベル | 説明 | 該当キャラクター |
|--------|------|------------------|
| `member` | 組織の一員 | 佐藤浩二、林健二・偽 |
| `system_aligned` | 直接の関係はないがシステム維持を優先 | 工藤真司 |
| `unknowing_connection` | 知らずに組織と繋がっている | 山田美咲 |
| `none` | 関係なし | その他一般市民 |
